# Support

## Where to Ask for Help

- Usage questions and troubleshooting: GitHub Discussions (if enabled) or GitHub Issues using templates.
- Reproducible bugs: open a bug report with steps, expected behavior, and logs.
- Security vulnerabilities: do not file a public issue; follow `SECURITY.md`.

## What to Include

For faster help, include:

- OS + Node.js + npm versions
- exact command(s) run
- expected vs actual behavior
- logs, screenshots, and reproduction steps

## Scope

Supported:

- current `main` branch behavior
- released versions published from this repository

Best effort:

- older versions and custom forks
- environment-specific local setup issues
